## 字符串和常用数据结构
### 字符串
表示：用单个或单个字符，用单or双引号包围起来
```python
s1 = 'hello, world~!'

# 三个单or双引号可以折行
s2 = '''
hello,
world!
'''
```

**反斜杠**
- 表示转义
- 表示八进制、十六进制字符、Unicode字符编码表示字符
- 不希望表示转义 加r
```python1
s1 = '\'hello, world!\''  # ’hello,world!'
s2 = '\n\\hello, world!\\\n'  # \hello,world!\ 
s3 = '\141\142\143\x61\x62\x63'  # abcabc
s4 = '\u9a86\u660a'  # 骆昊
```
<br/>

**字符串运算**

- 字符串的拼接
```python
s1 = 'hello ' * 3  # hello hello hello
s2 = 'world'
s1 += s2  # hello hello hello world
```
<br/>

- 使用in和not in判断是否包含另一个字符串
```python
print('ll' in s1)  # True
print('good' in s1)  # False
```
<br/>

- 切片运算：用\[]和\[:]取出某个字符或某些字符
```python
str2 = 'abc123456'

print(str2[2:5]) # c12
print(str2[2:]) # c123456
print(str2[2::2]) # c246
print(str2[::2]) # ac246
print(str2[::-1]) # 654321cba
print(str2[-3:-1]) # 45
```
